How it Works
Each gameplay transaction includes a small bit of information from a dataset, which is uploaded to the event logs of a contract on the Base chain.

As players make hundreds of thousands of transactions, they collectively upload large datasets to the blockchain.

ROADMAP

MILK PROTOCOL

STAGE 1

TEAM-MODERATED FILE UPLOADS

File uploads are curated and managed by the team.

STAGE 2

GOVERNANCE-DRIVEN FILE UPLOADS

Community members vote to approve files for blockchain uploads.

STAGE 3

DECENTRALIZED FILE UPLOADS

A fully decentralized process: users autonomously select and upload files to the blockchain, creating a fully decentralized, self-sustaining Machine Learning Library of Knowledge (MILK).

MILK GAME

STAGE 1

PETS → PLANETS

Players mine MILK by sending their pets to various planets.

STAGE 2

PETS+EQUIPMENT → PLANETS+PLACES

Equipment for pets and diverse planetary locations, enhancing strategy and gameplay depth.

STAGE 3

PETS RAIDS AND DEFENSE → PLANETS DEVELOPMENT

Tower Defense mechanics and social features, fostering collaborative and competitive gameplay.
